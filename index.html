How to Connect Google Sheets

1
Create a Google Sheet

Go to sheets.google.com → New spreadsheet. Name it "Attendance".

2
Add header row

Row 1: Name | Roll No | Role | Department | Status | Time | Confidence | Date

3
Open Apps Script

In your Sheet: Extensions → Apps Script. Paste the code from scripts/google-apps-script.js.

4
Deploy as Web App

Click Deploy → New deployment → Web App. Set access to "Anyone". Copy the URL.

5
Paste URL here

Paste the Web App URL in the field on the left. Save settings.

6
Copy Sheet ID

From your sheet URL, copy the ID and paste it in the Sheet ID field above.
